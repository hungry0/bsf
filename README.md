# bsf_copy
从零开始抄一个bsf游戏引擎

## 为什么是这个游戏引擎

| 游戏引擎  | 代码行数 |   功能    | 有生之年可抄袭的可能性 |
| :-------: | :------: | :-------: | :--------------------: |
|    UE     |  1200w   | * * * * * |           0            |
|   Unity   |  1100w   |  * * * *  |           0            |
|  Urho3D   |   180w   |           |           *            |
|   Orge    |          |           |                        |
| cocos2d-x |   100w   |    * *    |         * * *          |
|    bsf    |   44w    |  * * * *  |        * * * *         |

bsf抛开商用引擎的历史包袱，架构前卫，常见功能齐全，值得一抄。
